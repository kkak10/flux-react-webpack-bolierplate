# Sass 폴더 구조 설명.

 - base
 - compoents
 - helpers
 - layout
 - pages
 - vendor
 - main.scss

## Base
reset css나 기본 Style을 정의.

## Compoents
재사용 가능한 Compoent Style을 정의하는 폴더.

## Helpers
Variable, Mixin, Function등등을 담는 폴더.

## Layout
페이지의 기본 구조 혹은 Layout에 대한 Style을 정의.

## Pages
각 페이지에 해당하는 Style을 정의하는 폴더.

## vendor
외부 라이브러리를 담는 폴더.

## main.scss
다른 모든 scss파일을 import하는 Entry Point.
